Auth Service

Own login request validation and session issuance behind the OpenAPI contract.

contract.auth.login

Authenticates credentials and returns the declared session response.

Consumers: ru.frontend

Inputs

request.body.emailstring · required

Email value from the auth UI.

request.body.passwordstring · required

Password value from the auth UI.

Outputs

200.sessionAuthSession

Authenticated session payload defined by the OpenAPI contract.

Error Contract

401.INVALID_CREDENTIALSCredentials are syntactically valid but authentication fails.

Return the declared error response; do not issue partial sessions or fallback identities.
