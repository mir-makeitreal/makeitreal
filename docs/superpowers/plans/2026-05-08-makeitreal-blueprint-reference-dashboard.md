# Make It Real Blueprint Reference Dashboard Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Redesign the generated Make It Real dashboard so the Blueprint reads like API/SDK documentation while Kanban remains compact runtime state.

**Architecture:** Keep the dashboard static and dependency-free. Extend `src/preview/preview-model.mjs` with a structured `blueprint` document model derived from existing PRD/design-pack/board artifacts, then rewrite `src/preview/render-dashboard-html.mjs` to render a docs-first layout with a compact status rail. Preserve the read-only dashboard boundary.

**Tech Stack:** Node ESM, static HTML/CSS/JS generated by the existing preview renderer, Node test runner.
