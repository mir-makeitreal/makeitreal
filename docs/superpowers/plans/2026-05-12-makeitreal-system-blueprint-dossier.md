# Make It Real System Blueprint Dossier Implementation Plan

> **For agentic workers:** This plan is self-contained. Execute it task-by-task from the repository files and commands listed below; no external skill, plugin, or prior chat context is required.

**Goal:** Turn the generated Make It Real Blueprint dashboard and live wiki from a single-surface reference page into a system-level architecture dossier that explains modules, contracts, dependency flow, call stack, signal flow, ownership, and evidence before implementation begins.

**Architecture:** Keep the preview static, read-only, dependency-free, and generated from canonical artifacts. Add a `systemDossier` document model in `src/preview/preview-model.mjs` that derives a multi-module architecture view from `design-pack.json`, then make `src/preview/render-dashboard-html.mjs` render that model with an API/SDK-documentation information architecture. Reuse the same dossier model shape in `src/wiki/live-wiki.mjs` so verified work updates the live wiki with the same module/contract map.

**Tech Stack:** Node ESM, static HTML/CSS/JS generated by the existing preview renderer, Make It Real design-pack artifacts, Node test runner.

---

## Current Problem

The current renderer still has a single-primary-surface bias:

- `src/preview/render-dashboard-html.mjs` uses `primarySurface()` and `primaryModule()` at the page level.
- `referenceTitle()` can let a single function or request dominate the first viewport.
- `Usage Contract` is shown before the system map, which makes the dashboard look like one function doc instead of a feature architecture dossier.
- Architecture, sequence, and call stack are present, but they are rendered as isolated sections instead of an integrated overview that answers: "How does this new feature connect to the existing system?"
- The live wiki in `src/wiki/live-wiki.mjs` also centers one primary module and does not yet include a system map or dependency matrix.

The new target is not a full project-management UI. The dashboard remains read-only observability. State changes still happen only through Claude Code commands, hooks, and gates.

## Design Contract

The dashboard must answer these questions in the first two scrolls:

1. What feature/system slice is being built?
2. Which responsibility units are involved?
3. Which module owns which paths and public surfaces?
4. Which contracts can cross module boundaries?
5. Which existing/planned module calls which other module?
6. What is the signal or sequence flow?
7. What is the expected call stack?
8. What evidence proves completion?
9. What is the current run state, without making harness internals the primary content?

The browser page is still read-only:

- Allowed: copy the next Claude Code command.
- Allowed: show Kanban as compact runtime state.
- Forbidden: approve, launch, retry, reconcile, or mark Done in the browser.
- Forbidden: call `makeitreal-engine` from browser JavaScript.

```js
function uniqueText(values = []) {
  return [...new Set(values.filter((value) => typeof value === "string" && value.trim().length > 0))];
}

function contractIndex(contracts = []) {
  return new Map(contracts.map((contract) => [contract.contractId, contract]).filter(([contractId]) => Boolean(contractId)));
}

function contractSummary(contract, fallbackReason = "Declared boundary contract.") {
  if (!contract) {
    return fallbackReason;
  }
  if (contract.kind === "none") {
    return contract.reason ?? fallbackReason;
  }
  return `${contract.kind} contract at ${contract.path}`;
}

function architectureNodeLabel(nodes = [], id) {
  const node = nodes.find((candidate) => candidate.id === id || candidate.responsibilityUnitId === id);
  return node?.label ?? node?.responsibilityUnitId ?? id;
}

function modelDependencyEdges({ designPack, contracts }) {
  const contractsById = contractIndex(contracts);
  const architecture = designPack.architecture ?? {};
  const architectureEdges = (architecture.edges ?? []).map((edge) => ({
    from: edge.from,
    fromLabel: architectureNodeLabel(architecture.nodes ?? [], edge.from),
    to: edge.to,
    toLabel: architectureNodeLabel(architecture.nodes ?? [], edge.to),
    contractId: edge.contractId ?? null,
    contractKind: contractsById.get(edge.contractId)?.kind ?? "contract",
    allowedUse: contractSummary(contractsById.get(edge.contractId), "Architecture dependency.")
  }));

  const importEdges = (designPack.moduleInterfaces ?? []).flatMap((moduleInterface) =>
    (moduleInterface.imports ?? []).map((dependency) => ({
      from: moduleInterface.responsibilityUnitId,
      fromLabel: moduleInterface.moduleName ?? moduleInterface.responsibilityUnitId,
      to: dependency.surface ?? dependency.contractId,
      toLabel: dependency.surface ?? dependency.contractId,
      contractId: dependency.contractId ?? null,
      contractKind: contractsById.get(dependency.contractId)?.kind ?? "import",
      allowedUse: dependency.allowedUse ?? contractSummary(contractsById.get(dependency.contractId), "Declared module import.")
    }))
  );

  const seen = new Set();
  return [...architectureEdges, ...importEdges].filter((edge) => {
    const key = `${edge.from}|${edge.to}|${edge.contractId}|${edge.allowedUse}`;
    if (seen.has(key)) {
      return false;
    }
    seen.add(key);
    return true;
  });
}

function modelContractMatrix({ contracts, moduleInterfaces }) {
  return contracts.map((contract) => {
    const providers = moduleInterfaces
      .filter((moduleInterface) =>
        (moduleInterface.publicSurfaces ?? []).some((surface) => (surface.contractIds ?? []).includes(contract.contractId))
      )
      .map((moduleInterface) => moduleInterface.moduleName);
    const consumers = moduleInterfaces
      .filter((moduleInterface) => (moduleInterface.imports ?? []).some((dependency) => dependency.contractId === contract.contractId))
      .map((moduleInterface) => moduleInterface.moduleName);
    return {
      contractId: contract.contractId ?? contract.kind,
      kind: contract.kind,
      path: contract.path ?? null,
      summary: contractSummary(contract),
      providers: uniqueText(providers),
      consumers: uniqueText(consumers)
    };
  });
}

function modelSystemDossier({ prd, designPack, responsibilityUnits, contracts }) {
  const moduleInterfaces = modelModuleInterfaces({ designPack, responsibilityUnits });
  return {
    title: prd.title,
    summary: prd.userVisibleBehavior ?? [],
    goals: prd.goals ?? [],
    modules: moduleInterfaces.map((moduleInterface) => ({
      responsibilityUnitId: moduleInterface.responsibilityUnitId,
      moduleName: moduleInterface.moduleName,
      owner: moduleInterface.owner,
      purpose: moduleInterface.purpose,
      owns: moduleInterface.owns,
      publicSurfaces: moduleInterface.publicSurfaces,
      imports: moduleInterface.imports
    })),
    dependencyEdges: modelDependencyEdges({ designPack, contracts }),
    contractMatrix: modelContractMatrix({ contracts, moduleInterfaces }),
    signalFlows: designPack.sequences ?? [],
    callStacks: designPack.callStacks ?? [],
    stateTransitions: designPack.stateFlow?.transitions ?? [],
    deliveryScope: {
      ownedPaths: uniqueText((designPack.responsibilityBoundaries ?? []).flatMap((boundary) => boundary.owns ?? [])),
      responsibilityUnitIds: uniqueText((designPack.responsibilityBoundaries ?? []).map((boundary) => boundary.responsibilityUnitId)),
      acceptanceCriteriaIds: (prd.acceptanceCriteria ?? []).map((criterion) => criterion.id ?? "AC")
    },
    designPatterns: [
      {
        name: "Contract-first responsibility boundary",
        rationale: "Adjacent modules may rely only on declared public surfaces and contract IDs."
      },
      {
        name: "Fail-fast contract mismatch",
        rationale: "Undeclared IO, imports, or fallback behavior must revise the Blueprint instead of being hidden in implementation."
      }
    ]
  };
}
```

```js
function renderModuleNode(module) {
  return `<article class="module-node">
    <header>
      <span>${escapeHtml(module.responsibilityUnitId)}</span>
      <strong>${escapeHtml(module.moduleName)}</strong>
    </header>
    <p>${escapeHtml(module.purpose ?? "Declared responsibility unit.")}</p>
    <div class="chip-row">
      <span>${escapeHtml(module.owner ?? "owner missing")}</span>
      <span>${(module.publicSurfaces ?? []).length} surface${(module.publicSurfaces ?? []).length === 1 ? "" : "s"}</span>

function renderSystemMap(dossier = {}) {
  const modules = dossier.modules ?? [];
  if (modules.length === 0) {
    return '<p class="empty">No modules declared.</p>';
  }
  return `<div class="system-map">
    ${modules.map(renderModuleNode).join("")}
  </div>`;
}

function renderDependencyMatrix(edges = []) {
  if (edges.length === 0) {
    return '<p class="empty">No cross-module dependencies declared.</p>';
  }
  return `<div class="dependency-matrix" role="table" aria-label="Dependency graph">
    <div class="matrix-row header" role="row">
      <div role="columnheader">From</div>
      <div role="columnheader">To</div>
      <div role="columnheader">Contract</div>
      <div role="columnheader">Allowed Use</div>
    </div>
    ${edges.map((edge) => `<div class="matrix-row" role="row">
      <div role="cell" data-label="From">${escapeHtml(edge.fromLabel ?? edge.from)}</div>
      <div role="cell" data-label="To">${escapeHtml(edge.toLabel ?? edge.to)}</div>
      <div role="cell" data-label="Contract"><code>${escapeHtml(edge.contractId ?? "none")}</code></div>
      <div role="cell" data-label="Allowed Use">${escapeHtml(edge.allowedUse ?? "")}</div>
    </div>`).join("")}
  </div>`;
}

function renderContractMatrix(rows = []) {
  if (rows.length === 0) {
    return '<p class="empty">No contracts declared.</p>';

  return `<div class="contract-matrix">
```

```js
function renderSystemMap(artifacts) {

  return `## System Map

${renderTable(["Module", "Responsibility Unit", "Owner", "Owns", "Public Surfaces"], interfaces.map((moduleInterface) => [
    moduleInterface.moduleName ?? moduleInterface.responsibilityUnitId,

    asCodeList(moduleInterface.owns ?? []),
    asCodeList((moduleInterface.publicSurfaces ?? []).map((surface) => surface.name))
  ]))}

function renderDependencyGraph(artifacts) {
  const contracts = new Map((artifacts.designPack.apiSpecs ?? []).map((spec) => [spec.contractId, spec]));
  const architectureRows = (artifacts.designPack.architecture?.edges ?? []).map((edge) => [
    edge.from,
    edge.to,
    edge.contractId ? `\`${edge.contractId}\`` : "None declared",
    contracts.get(edge.contractId)?.reason ?? contracts.get(edge.contractId)?.path ?? "Declared architecture edge"
  ]);
  const importRows = (artifacts.designPack.moduleInterfaces ?? []).flatMap((moduleInterface) =>
    (moduleInterface.imports ?? []).map((dependency) => [
      moduleInterface.moduleName ?? moduleInterface.responsibilityUnitId,
      dependency.surface ?? dependency.contractId,
      dependency.contractId ? `\`${dependency.contractId}\`` : "None declared",
      dependency.allowedUse ?? "Declared import"
    ])
  );

  return `## Dependency Graph

${renderTable(["From", "To", "Contract", "Allowed Use"], [...architectureRows, ...importRows])}
`;
}

function renderCallStacks(artifacts) {
  const stacks = artifacts.designPack.callStacks ?? [];
  return `## Call Stack

${stacks.length === 0 ? "None recorded" : stacks.map((stack) => `### \`${stack.entrypoint}\`

${asList(stack.calls ?? [])}`).join("\n\n")}
`;
}
```
