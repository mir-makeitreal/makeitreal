POST /books -> object

HTTP contract surface for POST /books.

Inputs

  • requestBody object · required

    Payload accepted by POST /books.

Outputs

  • 200 response object

    Successful response body defined by the OpenAPI contract.

Errors

  • 400.DECLARED_ERROR The implementation returns the declared 400 response. · Return only the declared OpenAPI error shape; do not invent fallback response bodies.
  • 500.DECLARED_ERROR The implementation returns the declared 500 response. · Return only the declared OpenAPI error shape; do not invent fallback response bodies.
  • CONTRACT_MISMATCH The implementation needs an input, output, status, or dependency not declared in the OpenAPI contract. · Fail fast and revise the Blueprint contract before implementation; do not hide the mismatch with fallback behavior.
Usage Example Declared surface only 
const requestBody = {
  query: "value"
};

const httpResponse = await fetch("/books", {
  method: "POST",
  headers: { "content-type": "application/json" },
  body: JSON.stringify(requestBody)
});

if (!httpResponse.ok) {
  throw new Error(`HTTP ${httpResponse.status}`);
}

const response = await httpResponse.json();
